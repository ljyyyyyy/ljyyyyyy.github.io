Go编程学习（一）

发表于 2019-07-22
本文字数： 1.6k 阅读时长 ≈ 1 分钟

这是我根据Go编程指南网站来学习Go的一些笔记。

Go中的包

  1. 在Go中，每个程序都是由包来构成的，程序运行的入口包是”main“。

  2. 包的导入使用import，如果需要导入多个包，可以用圆括号将包括起来，这在Go中称为”打包“。当然，你也可以编写多个导入语句，但是Go官方推荐使用打包的方式。

  3. 在导入了一个包之后，就可以使用其导出的名称来调用它。在Go中，首字母大写的名称标志着它是被导出的。Foo和FOO都是被导出的名称，foo是不会被导出的。

Go中的函数

  1. Go中的函数可以没有参数或者接受多个参数。==注意：类型要在变量名之后==

  2. 当两个或多个连续的函数命名参数是同一类型时，则除了最后一个类型之外，其他都可以省略。

  3. Go中的函数可以返回任意数量的返回值（卧槽！！！）

  4. Go中函数的返回值可以被命名，并且像变量那样使用。返回值的名称应当具有一定的意义可以作为文档使用。
